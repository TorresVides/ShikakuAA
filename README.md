# Solucionador de Shikaku

Proyecto del curso Análisis de Algoritmos.

## Descripción

Este proyecto implementa una interfaz por consola y un solucionador sintético para el rompecabezas Shikaku. El solucionador genera rectángulos candidatos para cada pista numérica y busca una partición válida del tablero usando fuerza bruta y programación dinámica con memoización.
